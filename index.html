Vérifier un ticket

Dépose ici le fichier PNG du ticket, tel que téléchargé depuis le bouton « TÉLÉCHARGER (PNG) ». Les données du ticket sont embarquées directement dans le fichier image ; elles seront extraites, puis le code d'authenticité sera recalculé et comparé.

Comment ça marche

Chaque ticket contient un code calculé à partir de ses données (joueur, mode, score, précision, série, référence temporelle) via une fonction de hachage, et ces mêmes données sont embarquées dans le fichier PNG lors du téléchargement. Déposer le fichier ici relit ces données directement dans l'image, recalcule le code localement et le compare à celui embarqué : s'ils correspondent, les chiffres affichés sont bien ceux générés par le jeu et n'ont pas été modifiés après coup.

Limite honnête — ce mécanisme tourne entièrement dans ton navigateur, sans serveur ni clé secrète cachée. Il protège contre une modification accidentelle ou une retouche rapide d'un score (image éditée, valeurs changées), mais quelqu'un qui lirait le code source de cette page pourrait en théorie fabriquer un ticket avec un code valide. Ce n'est donc pas une preuve cryptographique inviolable — plutôt un sceau d'intégrité qui décourage la triche occasionnelle. Notez aussi que certains outils de partage (réseaux sociaux, messageries) recompressent les images et peuvent effacer les métadonnées embarquées : dans ce cas la vérification échouera même si le ticket est authentique — utilisez le fichier téléchargé directement.
